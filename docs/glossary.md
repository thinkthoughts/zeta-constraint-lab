# Glossary

## Standard Terms

- **Riemann zeta function ζ(s)**: A complex function central to number theory, defined for complex numbers s.
- **critical line**: The line in the complex plane where Re(s) = 1/2.
- **zero**: A point s where ζ(s) = 0.
- **nontrivial zeros**: Zeros of ζ(s) located in the critical strip (0 < Re(s) < 1).
- **critical strip**: Region where 0 < Re(s) < 1.
- **spacing**: The difference between imaginary parts of consecutive zeros.

## Framework Notes

- **constraint**: A condition defining structured behavior (e.g., the critical line as a constraint boundary).
- **local structure**: Behavior of ζ(s) or zeros within a small neighborhood.
- **constraint boundary**: A region where behavior changes qualitatively (e.g., Re(s)=1/2).
- **perturbation**: Small change to parameters (e.g., shifting off the critical line).
